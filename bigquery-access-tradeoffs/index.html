The Call

MCP in Noor is the destination. Custom web app is the bridge. Hex is the placeholder — if anything.

MCP compounds with everything else being built and becomes part of what gets productized. The custom app is a legitimate 2-week de-risking step before wiring BigQuery into the orchestrator. Off-the-shelf only earns a spot as a 30-day bridge while the real thing is built — never as where we land.
